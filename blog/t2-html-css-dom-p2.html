What is grid based design?

A grid is a series of intersecting horizontal and vertical lines that serve as guides to place elements in a design. To put it in simple words, grids are a system for creating order among elements in a website.

Whats all the hype about responsive webpages?

Allows a user to see all aspects of a website on different devices. Basically it adjust and formats for the size of the device they are using.

What is semantic structure?

A semantic element clearly describes its meaning to both the browser and the developer. An example would be to take out the div and span tags and put more relavant ones such as form, table, and img.
